# 🎨 E-Commerce Frontend

## 📌 Project Overview

This is the frontend of the E-Commerce application built using **React.js**. It provides a user-friendly interface for browsing products.

---

## 🚀 Features

* User-friendly UI
* Authentication (Login/Register)
* Responsive design (Mobile + Desktop)

---

## ⚙️ Setup Steps

### 🔧 Prerequisites

* Node.js (v16+)
* npm or yarn

---

### 🖥️ Local Setup

1. Clone the repository:

git clone https://github.com/kondamahesh1250/ecommerce.git
cd ecommerce-frontend

2. Install dependencies:

npm install

3. Start the application:

npm run dev

👉 App will run at:

http://localhost:5173
